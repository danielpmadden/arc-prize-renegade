# 0006: Deterministic Percept Formation

> **Document role: engineering journal.** This records Pass 6 choices, not a claim of official status.

Percepts are separate from measurements because a measurement reports a property,
whereas a percept preserves an explicitly formed structural unit. A connected
region is not automatically an object; no semantic concept is assigned.
Four-directional connectivity is the smallest explicit adjacency rule for this
bounded implementation and excludes diagonal contact. Formation capabilities
remain explicit so selection and interpretation are never hidden. The playground
is an architectural inspection tool that renders supplied input, records, and a
trace. Attributes, relationships, semantic association, interpretation,
reasoning, learning, and ARC handling are deferred.
