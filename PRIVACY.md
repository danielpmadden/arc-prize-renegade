# Privacy Statement

## Repository data collection

Renegade itself does not collect user data. No telemetry is intentionally
built into Renegade at this stage.

## Platform data

This repository is hosted on GitHub. GitHub may collect and process data in
connection with repository access and use under GitHub's own policies. Those
practices are outside Renegade's control.

## Future changes

Future Renegade software may include optional telemetry. If that is ever
implemented, it will be documented separately before relying on it.
